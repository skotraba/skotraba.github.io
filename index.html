About Me

I went to college to study Chemistry, but after working at an internship as an analytical chemist I found that I still wasn't fulfilled in my career and made the decision to learn how to program. I started with Python to learn general programming. I then moved to C++ to understand more about memory and how software works. I have comfortably transitioned into learning web development and I truly enjoy its variety of creative prospects. I love being able to use creativity and logic to create things that I am proud of. I am passionate about learning and programming gives me something greater than a job.

Projects

Color Picker

A window that shows a color wheel and allows the user to select a color. Based on the user selection, two other colors are generated to show a triadic color scheme. A useful program for artist and designers. The values were calculated using properties of an equilateral triangle. Written in C++ using the SFML library for graphics.

Library Management

A console command program that manages books in a library. A librarian can log in with an admin password and add new books to the library, check out books and see what is unavailable. A student can create their own username and password and see which books are available. The books are saved in different text files depending on their availability. Written in C++.

Recipe Calculator

A console command program that allows the user to input recipe info. The user is prompted to add ingredients listing calories and weight. After adding recipes the user can view the recipe or calculate total calories and calories based on serving size or weight. Written in Python.

Color Guess Game

A random color is picked from an array of nine different colors. The Red, Green, Blue value is displayed and the user can click on the circles to guess which color matches the values presented. If they guess wrong, the incorrect option dissapears and they can try until they get it correct. The play again button refreshes the browser to restart the game.
